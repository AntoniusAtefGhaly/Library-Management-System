# 📚 Library Management System (Learning Project)

This repository is a hands-on learning project built to transition theoretical knowledge into practical experience. It is a full-stack Library Management System designed to explore, implement, and master modern development practices.

## 🎯 Educational Goals & Applied Core Concepts
This project serves as a testing ground for implementing the following technologies and architectural patterns:

- **Frontend Framework:** Building responsive UI and handling state management using **Angular 19**.
- **Backend Architecture:** Structuring robust and scalable .NET 8 APIs using **Clean Architecture** principles (separating Domain, Application, Infrastructure, and Presentation layers).
- **Automation & DevOps:** Setting up real-world **CI/CD pipelines** using GitHub Actions for automated building, testing, and continuous deployment.
- **RESTful API Best Practices:** Implementing advanced server-side features like dynamic filtering and standardized pagination (PageNumber/PageSize).

This repo is continuously evolving as new concepts are learned and integrated.
